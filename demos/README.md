# About the game

Goal of the game: move the red block left and right to avoid black obstacles and falling down from the platform

<br/>

Control: 
* A: left
* D: right 

****
## Updates: 

### E08-Game Over
* game over: if the red block falls down from the platform or hits an obstacle, the game will be reset after 1 second

### E09-Winning Levels
* complete level UI: if you won, the UI will show "LEVEL COMPLETE"
* next level: if you completed level 1, level 2 scene will be loaded

### E10-Finishing Up
* menu: the welcome page of the game, and a button to start level 1
* credit: the final page of the game after you complete all levels, and a quit button to leave the game
